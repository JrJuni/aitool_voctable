프로젝트 : 고객 VOC 관리용 DB 파이프라인 설계 및 구현

# Overview
- 생성된 스키마 검증, 검증용 UX/UI 설계
- 로그인 시스템, 권한 관리 체계 개발

# Detail
1. 스키마 및 구조 검증
- schemas.py 기반 전체 테이블/필드 검토, 한글 주석 추가로 가독성 향상
- User/Company/Contact 등 주요 엔티티별 Pydantic 스키마 확정
- Enum(상태, 우선순위, 감사로그 액션) 정의 및 활용

2. 로그인/권한 시스템 개발
- Streamlit 기반 로그인 폼 구현, 세션 토큰 발급 및 상태 관리
- RBAC(권한 레벨 0~5) 적용: 가입/승인/비활성화/비밀번호 변경 등 권한별 분기
- 회원가입/비밀번호 초기화/관리자 승인 플로우 설계 및 UI 반영
- 관리자(레벨4+)만 비밀번호 초기화/승인 가능, HR 계정(고정) 생성

3. 프론트엔드 UX/UI 설계
- Streamlit 탭 구조: 로그인/회원가입/비밀번호 초기화/메인 VOC 테이블
- 사용자 정보, 설정, 로그아웃 등 상단 UI 배치 및 세션 연동
- 권한별 접근 제어(탭/버튼 노출), 에러/성공 메시지 처리

4. 감사로그/보안
- 사용자 관리 작업(생성/수정/삭제/비밀번호 변경) 감사로그 설계
- 비밀번호 정책/초기화(0000), 세션 토큰 관리, URL 파라미터 보호

5. 파일/코드 구조 정비
- backend/app/schemas.py: 한글 주석 포함 스키마 정리
- frontend/streamlit_app.py: 인증/권한/세션/탭 UI 코드 작성
- voc_requirements.txt: 요구사항/운영/보안 체크리스트 보강
